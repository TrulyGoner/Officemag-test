Верстальщик
	отличные знания HTML и CSS, начальные знания PHP

	полное соответствие вёрстки макету
		проверяется наложением снимка на макет
			отличия могут быть только из-за разной отрисовки в разных браузерах
			например, шрифты разные программы рисуют по-своему

	кроссбраузерность
		открытие вёрстки в разных браузерах не влияет заметно на результат
		отличия, как было упомянуто в прошлом пункте, связаны с отрисовкой текста
		или, если сильно повезёт, с ошибками в браузерах
			в случае ошибки браузера, в решении ожидается ссылка на задачу в багтрекере соответствующего браузера

	грамотность
		классы называются существующими английскими словами
		русский текст преимущественно без ошибок

Технолог
	Отличные знания HTML, CSS и JS, хорошие знания PHP

	адекватная методология
		код выглядит однородно
			написанным одним человеком, по единым принципам
			код легко изменяем

	не усложняет
		использует технологию, соответствующую задаче
			придумывает как эффективно обходиться стилями, вместо использования скрипта
			и использует скрипт там, где стили помешают расширению

	самостоятельность
		может придумать, как будут выглядеть состояния элементов
		и сделает это гармонично
	
	оптимизация
		код написан эффективно и оптимально
		картинки пожаты
